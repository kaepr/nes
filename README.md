# NES Emulator

NES Emulator made in Clojure for learning purposes. Work in progress.

Project bootstrapped using [deps-new](https://github.com/seancorfield/deps-new)

## Dependencies
- [clojure2d](https://github.com/Clojure2D/clojure2d)
